# 📊 E-Commerce Customer Analytics & Interactive Dashboard

## 🚀 Project Overview

The **E-Commerce Customer Analytics & Interactive Dashboard** is a Data Analytics project developed to analyze customer purchasing behavior, identify business insights, and create an interactive dashboard for decision-making.

This project demonstrates the application of **Python, Data Analytics, Machine Learning, and Business Intelligence** techniques to solve real-world business problems.

---

## 🎯 Objectives

* Analyze customer purchasing patterns.
* Define and track Key Performance Indicators (KPIs).
* Perform customer segmentation using K-Means Clustering.
* Visualize insights through an interactive dashboard.
* Support business decision-making with data-driven insights.

## 📊 Key Performance Indicators (KPIs)

### 1. Total Revenue

Measures the total revenue generated from customer purchases.

### 2. Average Order Value (AOV)

Measures the average amount spent per order.

### 3. Total Customers

Represents the number of unique customers.

### 4. Category-Wise Revenue

Analyzes revenue contribution from each product category.

### 5. Customer Segmentation

Groups customers based on purchasing behavior.

---

## 🔍 Customer Segmentation

The project uses **K-Means Clustering** to segment customers based on:

* Purchase Amount
* Order Count

### Customer Groups

* High-Value Customers
* Medium-Value Customers
* Low-Value Customers
* Potential Loyal Customers

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Ecommerce_Analytics_Dashboard.git
```

### Navigate to Project Folder

```bash
cd Ecommerce_Analytics_Dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

### Step 1: Generate Dataset

```bash
python generate_dataset.py
```

### Step 2: Perform Data Analysis

```bash
python analysis.py
```

### Step 3: Perform Customer Segmentation

```bash
python customer_segmentation.py
```

### Step 4: Launch Dashboard

```bash
streamlit run app.py
```

Dashboard will open automatically in your browser:

```text
http://localhost:8501
```
